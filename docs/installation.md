# Installation

## Stable release

To install Automatic Linux Network Repair, run this command in your terminal:

```sh
uv add automatic_linux_network_repair
```

Or if you prefer to use `pip`:

```sh
pip install automatic_linux_network_repair
```

## From source

The source files for Automatic Linux Network Repair can be downloaded from the [Github repo](https://github.com/JohnDoe6345789/automatic_linux_network_repair).

You can either clone the public repository:

```sh
git clone git://github.com/JohnDoe6345789/automatic_linux_network_repair
```

Or download the [tarball](https://github.com/JohnDoe6345789/automatic_linux_network_repair/tarball/master):

```sh
curl -OJL https://github.com/JohnDoe6345789/automatic_linux_network_repair/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd automatic_linux_network_repair
uv pip install .
```
